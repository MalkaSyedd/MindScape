# MindScape — Mood & Wellness Tracking Platform (Figma Prototype)

MindScape is a wellness-focused user experience designed entirely in Figma. It helps users track their mood, reflect through guided journaling, and build healthier daily habits. The design promotes mindfulness, emotional awareness, and positive behavior change through a calm, supportive interface.

## Purpose
This project was created as an interactive **concept prototype**, focusing on:
- UX/UI design principles
- Emotional accessibility and comfort
- Guided flow for healthier routines
- Simple navigation and low-effort logging

## Key Prototype Features
- Mood slider and day rating
- Guided journaling prompts
- Personalized activity suggestions
- Memory and photo reflection features
- Analytics showcasing emotional patterns
- Favorites section for uplifting activities
- Onboarding tutorial + navigation drawer
